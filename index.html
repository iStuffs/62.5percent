The magic number is 62.5%

The default browser user agent root `font-size` is 16px
So to have a base 10 math we can do a little math: 100% * 10 / 16 = 62.5%. 

      
  html {
    font-size: .625em; /* fallback IE8+ */
    font-size: calc(1em * .625); /* IE9-IE11 math fixing. See http://bit.ly/1g4X0bX */
  }
